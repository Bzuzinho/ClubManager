# Como iniciar o backend e frontend

Abra dois terminais e execute:

## Terminal 1: Backend
```bash
cd backend
php artisan serve --host=0.0.0.0 --port=8000
```

## Terminal 2: Frontend
```bash
cd frontend
npm run dev -- --host 0.0.0.0 --port 5173
```

Pronto! Cada serviço ficará rodando em seu terminal.
