## Estrutura do Projeto

### Backend (Laravel)
- Framework: Laravel
- Linguagem: PHP
- Base de Dados: Configurada em `config/database.php`
- API: Rotas em `routes/api.php`

### Frontend (React + TypeScript)
- Framework: React + TypeScript
- Build: Vite
- Configuração: `vite.config.ts`
