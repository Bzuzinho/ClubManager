Ficha Utilizador:
Deve ser uma página que os administradores podem consultar e editar todos os campos
Deve ser uma página que pode ser acedida pelos Encarregados de educação
No campo onde aparece o “educando”, quando se clica nele, deve abrir a ficha do educando,
dando acesso ao EE de alterar ou consultar todos os dados ( que um utilizador normal sem ser
admin poderia alterar)
Deve ser uma página que terá 4 separadores ( Pessoal, Financeiro, Desportivos, Configuração):
Separador 1 - Pessoal:

Se o campo “menor” for positivo deve aparecer para escolher o campo “encarregado_educacao” de onde
será escolhido um utilizador já existente com o tipo de “tipo_membro” “Encarregado de Educação”, caso
contrario deve ficar escondido. O utilizador que for escolhido para este campo de
“encarregado_educação” de herdar para o seu campo “educando” o utilizador que o adicionou.
​
Se o campo “menor” for negativo e o campo “tipo_membro” tiver um tipo “encarregado_educacao” deve
mostrar o campo “educando” para poder ser escolhido um utilizador que tenha o campo “menor” positivo.
Se o campo “tipo_membro” tiver numa das opções escolhidas o tipo “atleta”, deve colocar o separador
“Desportivo” visivel, caso contrário deve estar escondido.

Sub Separador - Registo Presenças
​
Deve conter a tabela com o registo de presenças que dizem respeito ao atleta

Sub Separador - Resultados
​
Deve conter a tabela com o registo de resultados que dizem respeito ao atleta

Sub Separador - Treinos
​
Deve conter a lista de treinos a que o atleta foi adicionado
​
Deve permitir abrir e consultar o treino
​
Deve permitir que o atleta selecione o treino como concluído
​
Deve permitir que o atleta corrija os metros finais

Será nesta página que é gerido os acessos de utilizador à plataforma, o email de acesso e a palavra
pass.
O campo email_utilizador deverá ser igual na criação do utilizador ao campo email, para no futuro poder
ser editado.
Deve ser possível reenviar a recuperação de palavra pass
Será aqui que se escolhe o tipo de perfil de utilizador e as permissões de acesso
